# Projet-Final
Classification d'images (Personnages Nintendo)

L'objectif du projet est de réaliser une classification d’images permettant de reconnaître 15 personnages de l’univers Nintendo:

- Mario
- Link
- Pikachu
- Fox
- Toad
- Yoshi
- Harmonie
- Daisy
- Waluigi
- Tiny Kong
- Luigi
- Captain Falcon
- Phoenix Wright
- Donkey Kong
- Professor Layton
